A culture, a mindset, and a set of practices to deliver better software, faster.

Definition

DevOps is a combination of Development and Operations. It focuses on collaboration, automation, integration, and continuous delivery between software developers and IT operations teams.

Goals of DevOps
  • Faster time to market
  • Improved collaboration
  • Continuous Integration & Delivery (CI/CD)
  • Better product quality
  • Automated infrastructure
DevOps Tools

Some popular tools used in DevOps pipelines:

  • Git – Version Control
  • Jenkins – CI/CD Automation
  • Docker – Containerization
  • Kubernetes – Orchestration
  • Ansible – Configuration Management
  • AWS / Azure – Cloud Platforms
DevOps Lifecycle

The DevOps lifecycle includes:

Plan Develop Build Test Release Deploy Operate Monitor
